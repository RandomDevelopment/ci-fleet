# Architecture decision records

This directory records durable project decisions. Each accepted ADR is authoritative for its subject until a later ADR supersedes it.

| ADR | Decision | Status |
| --- | --- | --- |
| [0001](0001-actions-scale-set-client.md) | Use the GitHub Actions scale-set client | Accepted |
| [0002](0002-public-engine-private-configuration.md) | Keep the engine public and real delivery configuration private | Accepted |
| [0003](0003-repository-markdown-is-canonical.md) | Repository Markdown is the documentation source of truth | Accepted |
| [0004](0004-adopt-delivery-fleet-name.md) | Adopt `delivery-fleet` as the umbrella project name | Accepted; migration deferred |
| [0005](0005-one-command-target-host-setup.md) | Provide one-command target-host setup | Accepted |

Changes to an accepted decision require a pull request that updates the relevant ADR or adds a superseding ADR.
